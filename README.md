# Job Portal

A modern job portal platform built with Next.js and Supabase. The application allows users to browse job opportunities, apply for jobs, and manage applications through a dedicated admin dashboard.

## Features

### User Features

* User authentication and authorization
* Browse available job listings
* Filter and search jobs
* View job details
* Submit job applications

### Admin Features

* Secure admin authentication
* Manage job listings
* Create new job posts
* View submitted applications
* Monitor platform activity

## Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* Supabase Authentication
* Supabase Database

## Installation

```bash
git clone <repository-url>

npm install

npm run dev
```

## Project Goals

The goal of this project is to provide a simple and efficient platform for connecting job seekers with available opportunities while giving administrators full control over job management.
